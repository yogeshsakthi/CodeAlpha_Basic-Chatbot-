# 🤖 Basic Chatbot

A simple rule-based chatbot built using Python. The chatbot responds to a few predefined user inputs and continues the conversation until the user types **"bye"**.

## 📌 Project Description

This project demonstrates the basics of Python programming using:

- Functions
- Loops
- Conditional statements (`if`, `elif`, `else`)
- User input and output

The chatbot recognizes a few predefined messages and replies accordingly.

## ✨ Features

- Greets the user.
- Responds to:
  - `hello`
  - `how are you`
  - `bye`
- Displays a default message for unknown inputs.
- Runs continuously until the user exits.

## 🛠️ Technologies Used

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python chatbot.py
```
